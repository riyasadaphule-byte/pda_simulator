# PDA Simulator

## Overview

This project is a Pushdown Automata (PDA) simulator developed to understand the working of PDAs in theory of computation. It supports both deterministic and non-deterministic PDAs.

## Features

* Supports DPDA and NPDA
* Step-by-step execution of input
* Next button to move through transitions
* Back button to go to previous steps
* Displays stack at each step

## Working

The simulator processes the given input string step by step. At each stage, it shows the current state, remaining input, and stack content. The next and back buttons help in tracing the transitions clearly.

## Technologies Used

* HTML
* CSS (used inside the HTML file)
* JavaScript (used for logic and transitions)
